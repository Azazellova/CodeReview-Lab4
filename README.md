## Основные изменения (FIX_ME)

### 1. Меню выбора

* Выведено информативное меню вместо набора чисел.
* Добавлена проверка корректности выбора (только числа 0–6).

### 2. Ввод данных с консоли

* Для `Point`, `StraightLine` и `Student` добавлен полноценный ввод с клавиатуры.
* Координаты точек проверяются на допустимый диапазон (-10000 … 10000).
* Оценки студентов проверяются на диапазон 2–5, с завершением ввода `-1`.

### 3. Работа с массивами оценок

* Исправлено некорректное копирование массивов: теперь используется `clone()`.
* Обработан случай null или пустого массива.

### 4. Класс `Student`

* Удалено поле `gpa`; средний балл вычисляется динамически через `getAverage()`.
* Добавлена валидация имени и оценок при создании студента.
* Метод `toString()` обновлён для корректного вывода.
* Добавлена проверка на отличника (`isExcellent()`).

### 5. Класс `StraightLine`

* Исправлена обработка `null` при создании линии.
* Копирование координат при изменении начала и конца линии, чтобы избежать ссылочной зависимости.

### 6. Класс `Point`

* Удалён пустой конструктор.
* Добавлена валидация координат при установке.
* Название класса исправлено на соответствующее Java Code Style (`Point` вместо `KaordinatePoint`).

### 7. Класс `Validation`

* Добавлены методы для безопасного ввода чисел, координат и оценок.
* Проверка значений координат и оценок.
* Исправлена проверка выбора пункта меню.

### 8. Общие улучшения

* Добавлен метод `readArray(Scanner)` для ввода массива оценок.
* Исправлены ошибки, из-за которых задания 3.1 и 5.6 некорректно работали с пользовательским вводом и копированием данных.
